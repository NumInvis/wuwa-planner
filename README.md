# 鸣潮账号规划文档编辑器 (WuWa Planner)

一个用于生成鸣潮游戏账号分析和抽卡规划文档的 Web 工具，可以导出为 LaTeX 和 PDF 格式。

## 功能特点

- 📝 **引导式输入**: 按照标准文档结构分模块填写
- 🎨 **美观界面**: 现代化的 Web 界面，响应式设计
- 📄 **LaTeX 导出**: 生成标准格式的 LaTeX 代码
- 📑 **PDF 生成**: 一键编译生成 PDF 文档
- 🔧 **自定义专题**: 支持添加任意专题分析
- 🎯 **配队表格**: 自动生成属性配队表格

## 安装要求

- Python 3.7+
- Flask
- XeLaTeX (用于生成 PDF)

## 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/yourusername/wuwa-planner.git
cd wuwa-planner
```

2. 安装依赖
```bash
pip install flask
```

3. 运行程序
```bash
python web_editor.py
```

4. 在浏览器中访问 `http://localhost:5000`

## 使用方法

1. **基本信息**: 输入 UID 和作者名
2. **原文**: 填写用户提供的原始信息（资源、预算、偏好等）
3. **配队情况**: 填写六个属性的配队配置
4. **整体情况**: 输入整体评价
5. **逐队伍分析**: 逐个添加队伍分析
6. **专题模块**: 添加自定义专题（如"聚爆爱or震谐爱"）
7. **逐版本抽卡建议**: 添加各版本的抽卡建议
8. **最终推荐方案**: 输入最终结论
9. **其他**: 添加养成建议等

点击"生成 LaTeX"预览代码，或点击"生成 PDF"直接下载 PDF 文件。

## 文档结构

生成的文档包含以下部分：

- 原文
- 配队（表格形式）
- 整体情况
- 逐队伍分析
- 专题模块（可自定义）
- 逐版本抽卡建议
- 最终推荐方案
- 其他

## 配队格式

配队格式：`链数+角色名`，例如：
- `21赞+20菲+41守` - 2链赞妮+2链菲比+4链守岸人
- `61椿+散+维` - 6链椿+散华+维里奈
- `00守` - 0链守岸人
